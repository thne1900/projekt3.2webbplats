Vilka kan resa med färdtjänst?

För dig som går under lagen (1997:735) om färdtjänst, med någon form av funktionsvariation, som gör att du inte kan på egen hand förflytta dig eller ta dig fram med kollektivtrafiken. Denna webbplats riktar sig till de personer som är bosatta inom Kungälvs kommun. Som är i behov att kunna boka en färdtjänstresa för sina vardagssituationer, som tex ett sjukhusbesök. Ett mindre fordon används vid dessa resor.

Du som har rätt till färdtjänst ska uppfylla vissa krav:

  • Svårigheter i hög grad att på egenhand kunna förflytta dig.
  • Inte kunna använda kollektivtrafiken (ex buss eller tåg) pga. av din funktionsvariation.
  • Bosatt och skriven i Kungälvs kommun, det är via kommunen som ansökan görs kring färdtjänst.
  • Din funktionsvariation är bedömd till att vara bestående minst tre månader.

Färdtjänsttillstånd:

Utfärdas då kraven uppfylls och innehåller den tid du är beviljad att få rätt till färdtjänst. Det kan också finnas förskrivet speciella regler för ditt innehav, som att resorna enbart får utföras inom ett begränsat område eller ett limiterat antal resor som kan nyttjas. Du får då ett personligt färdtjänstkort. Samåkning kan ske med andra resenärer.

Ansökan:

En ansökan om färdtjänst görs genom att skicka in ansökningsblanketten till Kungälvs kommun via post eller email. Ett läkarutlåtande behöver bifogas med blanketten.
